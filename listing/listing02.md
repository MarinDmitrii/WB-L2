Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1
Отложенные функции могут читать и устанавливать именованные возвращаемые значения объемлющей функции.

defer используется для отложенного выполнения функции или метода до момента завершения текущей функции или до выполнения оператора return.
defer-ы складываются в стек (LIFO)
При использовании отложенных функций с defer, аргументы, передаваемые в эту отложенную функцию, вычисляются в момент выполнения отложенного вызова (при вызове defer), а не в момент фактического вызова функции

```
