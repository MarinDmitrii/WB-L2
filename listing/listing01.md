Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]
b - срез, образованный из массива a
Форма записи a[1:4] указывает, что из массива a будут взяты элементы с индексами с 1 по 3

```
